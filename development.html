Project Development

An overview of how the website was planned, designed, and developed through research, testing, and iteration.

This page provides an overview of how the website was planned, designed, and developed. The project followed a structured process that included research, data collection, design decisions, user testing, and iterative improvements.

1. Planning & Tools

The website was created using a modern website approach due to its ease of use, responsive design features, and support for interactive elements such as forms, maps, and embedded media.

Tools used: Google Forms (data), Google Sheets (analysis), Canva (infographics), and Google Maps.

2. Prototyping

Throughout development, several versions of the website were created. Early versions focused on layout and navigation.

3. Iteration

Later versions added data visualisation, interactive forms, and improved colour and font choices based on user feedback to create a modern, nature-inspired palette.

4. User Testing

User testing was conducted with a small group of peers who provided feedback on clarity, navigation, and visual design. Their suggestions led to improvements such as clearer headings, reorganised content sections, and the addition of a pledge form to increase engagement.
